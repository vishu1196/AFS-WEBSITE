## 🚀 How to Upload to GitHub

1. Go to [github.com](https://github.com) and create a free account
2. Click **"New Repository"**
3. Name it: `afs-website`
4. Make it **Public**
5. Click **"Create repository"**
6. Upload all files using **"Add file → Upload files"**
7. Drag and drop the entire folder contents
8. Click **"Commit changes"**

### Make it a Live Website (Free!)
1. Go to your repository → **Settings**
2. Scroll to **Pages** (left sidebar)
3. Under "Branch", select `main` → `/ (root)`
4. Click **Save**
5. Your site will be live at: `https://yourusername.github.io/afs-website/`
